Проект: Proekt (Android-приложение)

Описание
Приложение предназначено для управления данными пользователя, работы с профилем, аналитикой и основными разделами приложения. Проект использует Firebase для авторизации и хранения данных.

Основные возможности
- Авторизация и выход из аккаунта
- Работа с профилем пользователя (логин, аватар)
- Навигация по ключевым разделам приложения
- Интеграция с Firebase (Auth/Firestore)

Структура проекта
- app/ — исходный код приложения и ресурсы
- build/ — артефакты сборки
- gradle/ — Gradle-обвязка
- settings.gradle.kts, build.gradle.kts — конфигурация сборки

Требования
- Android Studio (актуальная версия)
- JDK 17+
- Gradle (идет вместе с Android Studio)
- Настроенный Firebase-проект (google-services.json)
