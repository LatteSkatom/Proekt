Инструкция по запуску мобильного приложения

1) Подготовка окружения
- Установите Android Studio.
- Убедитесь, что установлен JDK 17+.
- Проверьте наличие Android SDK и эмулятора (AVD).

2) Настройка Firebase
- В папке app/ должен быть файл google-services.json.
- Если файла нет, создайте проект в Firebase Console и добавьте приложение Android.
- Скачайте google-services.json и поместите его в app/.

3) Запуск приложения
- Откройте проект в Android Studio: File -> Open -> выберите папку Proekt.
- Дождитесь синхронизации Gradle.
- Запустите приложение: Run -> Run 'app' (или кнопка ▶).
- Выберите устройство: эмулятор или подключенный Android-девайс.

4) Сборка APK (по желанию)
- Build -> Build Bundle(s) / APK(s) -> Build APK(s).
- Полученный APK будет в папке app/build/outputs/apk/.

5) Решение проблем
- Если возникают ошибки сборки, выполните: Build -> Clean Project и Build -> Rebuild Project.
- Проверьте корректность google-services.json и доступность сети.
